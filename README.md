# Marketing Campaign Performance Dashboard

## Project Overview
This project is an interactive Power BI dashboard built to analyze marketing campaign performance using KPIs, funnel analysis, customer segmentation, and campaign-level insights.

## Key Features
- Marketing funnel analysis
- ROI analysis by campaign type
- Revenue distribution by marketing channel
- Customer segmentation analysis
- Interactive slicers and filters
- KPI cards for revenue, clicks, conversions, and ROI

## Tools Used
- Power BI
- Excel
- Data Visualization
- Marketing Analytics

## Insights
- Social Media campaigns generated the highest ROI.
- Email campaigns showed comparatively lower ROI.
- Customer conversion rates remained relatively balanced across segments.
